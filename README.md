# 馬積み駐車場管理アプリ / Mare Transport Yard Manager

Excelの「馬積み・駐車場管理表」を、スマホ・PCで使える駐車場マップ型の業務アプリにしたもの。
馬運車ごとに、積載馬・繁殖牝馬・子供の誕生日/性別・出発元・降ろし状態（未降ろし/降ろし中/降ろした）・
1頭ずつ/2頭OK・駐車位置を、画面を見ただけで判断できる。

- React / Next.js 14（App Router）+ TypeScript
- 保存はブラウザの localStorage（キー: `mare-transport-yard-data`）。サーバー不要。
- 初期データは元Excel「馬積テスト.xlsx」を再現（駐車3=MAU/2頭OK、2=HRC/1頭ずつ、1=KZN/降ろした）。

## 使い方（ローカル）

```bash
cd umazumi-yard
npm install
npm run dev
# ブラウザで http://localhost:3210 を開く
```

## スマホ・他PCから使う（静的書き出し）

```bash
npm run build
# out/ フォルダが生成される（静的HTML）。
# out/ を Vercel / GitHub Pages / Netlify などに置けば、スマホからURLで利用可能。
```

## 主な機能

- 駐車場マップ表示（駐車スペース + 予備スペース）
- 車両カード（駐車番号・車両コード・頭数・降ろし方式・積載馬・降ろし状況を色分け表示）
- 車両カードの「降ろす」ボタンで1頭ずつ降ろし
- 詳細モーダルで 未降ろし/降ろし中/降ろした を切替、2頭OKの車は「まとめて降ろす」
- 車両・馬の追加/編集/削除
- 馬コード一覧（積載中はハイライト、クリックで現在地を確認）
- クリア（確認ダイアログ付き、初期データ復元も可）

## 色分け

| 状態 | 色 |
| --- | --- |
| 未降ろし | 白 |
| 降ろし中 | 黄色 |
| 降ろした | 黒/ダークグレー |
| 2頭OK | 青系ラベル |
| 1頭ずつ | オレンジ系ラベル |
| 予備 | 薄いグレー |
| 未入力・注意 | 赤系 |
